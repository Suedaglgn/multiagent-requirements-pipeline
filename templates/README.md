# Output Templates & Formatting Requirements

The expected shape of each pipeline artifact. These structures are embedded inside the prompts
themselves; they are surfaced here so the RA/DE/extractor output contracts are easy to locate.

| Artifact | File | Where it is defined |
|----------|------|---------------------|
| Requirement list categories | `extractor_categories.md` | Extractor Enhanced/Advanced prompt |
| RA.md (SRS) structure | `RA_output_template.md` | RA Advanced prompt |
| DE.md structure | `DE_output_template.md` | DE Advanced prompt |

Level dependence:
- **Basic**: guidelines only, section lists are NOT enforced.
- **Enhanced**: section titles only.
- **Advanced**: section titles with detailed guidance (shown in these templates).
